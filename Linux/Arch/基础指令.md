# 基础指令

## cp

```bash
cp 源文件 目标文件
```

## ssh

```bash
ssh root@127.0.0.1
```

## scp

本地传输至远程:

```bash
scp /home/passenger/Document/a.c root@127.0.0.1:/home/passenger/a.c
```

远程传输至本地:

```bash
scp root@127.0.0.1:/home/passenger/a.c /home/passenger/Document/a.c
```

## Debian Service

启动服务:

```bash
service ssh start
```

开机自启动:

```bash
update-rc.d ssh enable
```

关闭开机自启动:

```bash
update-rc.d ssh disable
```
